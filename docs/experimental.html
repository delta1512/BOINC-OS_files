BOINC OS Experimental

BOINC OS Experimental is a line of BOINC OS spins made for niche tasks and systems that are not suited to the mainstream distros. Many of these spins are community projects and I thank all those involved.

BOINC OS Installer

This is a tool used to install BOINC OS automatically to a USB or hard drive from your host OS. It intends to be a multi-platform GUI program that allows you to instantly install any BOINC OS spin straight to your USB in one click.

Status: Development not started

BOINC OS PXE Boot

PXE network boot is a method of getting a bootable image to a network of computers easily without having to install anything on them initially. This is typically used in enterprise style networks and this BOINC OS spin intends to take advantage of this concept. It will allow you to run a PXE server that allows you to install or simply run BOINC OS on a large number of computers.

Status: Development in progress

BOINC OS for RPI

A BOINC OS graphical spin for the Raspberry Pi. Aim of this OS to be simply flashed to an SD card and switched out with a typical image when you wish to compute.

Vbox TmpOS Spin

BOINC OS as a temporary OS, i.e, the OS is entirely in RAM. This will be for users who want speed when starting up or using the system but are not concerned with maintaining their progress. This is also the perfect distro for using in conjunction with Virtualbox.

Feel Like Contributing?

All BOINC OS experimental spins are open for the public to work on - if you feel up to the challenge of course. The following are some ways you could help the project:

Tackle one of them

Have a go at trying to make one of these happen! To get started, contact the main developer at boincosdelta@gmail.com so you can get kitted out with all the information you need. If you run into issues during development, feel free to post a GitHub issue and the community will help you out.

Help out a project in progress:

Help out a member already trying to tackle a project! Any relevant contact information for members will be posted on the above cards under the “Author” title.

Think of a new spin

New ideas are always accepted! If you have an idea to add to the experimental versions, put up a GitHub issue or email the main developer at boincosdelta@gmail.com.

Donate

There is no limit to generosity! Donations are always accepted, you can find out who you can donate to and what you can donate over on the contributors list.
